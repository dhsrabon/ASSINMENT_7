
# 📌 KeenKeeper

KeenKeeper is a modern web application designed to help users manage and maintain meaningful relationships. It allows you to track interactions, set connection goals, and stay in touch with friends more effectively.

---

## 🚀 Project Description

KeenKeeper helps users keep track of their personal and professional relationships by organizing friends, tracking communication history, and reminding users when to reconnect. It provides a clean and user-friendly interface for managing social connections.

---

## 🛠️ Technologies Used

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🌐 React Router DOM
- 🔥 Vite
- 📦 JavaScript (ES6)

---

## ✨ Key Features

1. 👥 **Friend Management**
   - Add and view multiple friends with detailed profiles.
   - Display friend cards with status, tags, and last contact time.

2. 📜 **Timeline Tracking**
   - Track interactions like calls, texts, videos, and meetups.
   - Automatically logs activities with date and type.

3. 📊 **Analytics Dashboard**
   - Visualize interaction data using charts.
   - Understand your communication patterns.

---

## 📱 Responsive Design

The application is fully responsive and works smoothly on:
- Mobile 📱
- Tablet 💻
- Desktop 🖥️
